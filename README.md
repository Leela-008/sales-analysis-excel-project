## Project Overview
This project involves creating an interactive sales dashboard in Advanced Excel to analyze sales and profit performance across categories, states, customers, and time. The dashboard is designed to provide clear business insights through dynamic and user-friendly visualizations.

## Key Features

Data cleaning and preparation for accurate analysis

Pivot Tables for summarizing sales and profit metrics

Interactive charts and slicers for category, state, customer, and time-based analysis

Monthly sales trends and profit analysis over time

Identification of top customers based on sales performance

## Tools & Techniques Used

Advanced Microsoft Excel

Pivot Tables & Pivot Charts

Excel formulas

Slicers and dynamic dashboards

Data visualization and reporting

## Outcome
The dashboard enables quick, data-driven decision-making by presenting sales and profit insights in a clear, interactive, and easy-to-understand format.
